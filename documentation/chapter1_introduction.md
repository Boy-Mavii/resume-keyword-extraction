# Chapter 1: Introduction

## 1.1 Background of the Study
In the modern recruitment landscape, the volume of resumes received for a single job opening can be overwhelming. Organizations often receive hundreds, if not thousands, of applications, making manual screening a time-consuming and error-prone process. Natural Language Processing (NLP) offers a solution by automating the extraction of key information from resumes, allowing recruiters to quickly identify the most qualified candidates based on specific keywords and skills.

## 1.2 Problem Statement
Manual resume screening is inefficient and subject to human bias. Recruiters often struggle to find relevant candidates within a large pool of applicants due to the unstructured nature of resumes, which come in various formats (PDF, DOCX) and layouts. There is a need for a robust, automated system that can parse these documents and extract meaningful keywords and entities accurately.

## 1.3 Objective of the Study
The primary objective of this project is to design and implement a Resume Keyword Extraction System using Natural Language Processing. The specific objectives include:
- To develop a system capable of extracting text from PDF and DOCX resume formats.
- To implement NLP techniques for Named Entity Recognition (NER) to identify names, organizations, and contact details.
- To create a keyword extraction algorithm to identify technical skills and professional competencies.
- To provide a user-friendly web interface for recruiters to upload resumes and view extracted data.

## 1.4 Significance of the Study
This system will significantly reduce the time and effort required for the initial stage of the recruitment process. It will help in:
- Improving the efficiency of HR departments.
- Reducing human bias in candidate shortlisting.
- Enabling better searchability of candidate profiles within an organization's database.

## 1.5 Scope of the Study
The project focuses on the extraction of keywords and entities from resumes in English. It supports PDF and DOCX formats. The extraction covers common professional fields but is most optimized for technical and corporate roles.
